# Things to set up on a new GNOME:

## Change to the Emacs keybinding theme
"Tweak Tool" -> "Keyboard and Mouse" -> "Key theme" ==> Emacs
## Swap Ctrl and Caps Lock
"Tweak tool" -> "Typing" -> "Ctrl key position" ==> "Swap Ctrl and Caps Lock"
## Show date & seconds on the clock
"Tweak Tool" -> "Top Bar" ==> Show date & show seconds
## Focus follows mouse
"Tweak Tool" -> "Windows" -> "Focus Mode" ==> "Sloppy"
## Make the Terminal not have menu-bars
"Edit" -> "Preferences" -> "General" ==> "Show menubar by default..."

# Other:
## Switch the default display for menubars etc.
xrandr --output HDMI3 --primary
